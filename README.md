# Task Management System

A scalable and collaborative task management backend built with Django and Django REST Framework.

## Features
- Custom user model
- Workspace-based collaboration
- Role-based access control
- Boards and task lists (Trello-like structure)
- Tasks with status, priority, and assignment
- Clean and modular app structure
- Pre-commit hooks with Black, Isort, and Ruff

## Tech Stack
- Python
- Django
- Django REST Framework
- PostgreSQL (planned)
- UV
- Pre-commit
- Black / Isort / Ruff
